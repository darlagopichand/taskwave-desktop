TaskWave is a desktop application designed to help professionals maintain flow and manage tasks efficiently. Unlike standard to-do lists, TaskWave focuses on "Deep Work" sessions, allowing users to organize tasks by priority and energy levels.

## üöÄ Features

* **Kanban Board View**: Drag-and-drop tasks between "To Do", "In Progress", and "Done".
* **Focus Timer (Pomodoro)**: Built-in timer to track deep work sessions.
* **Offline First**: All data is stored locally for privacy and speed.
* **Smart Reminders**: Desktop notifications for deadlines.
* **Dark/Light Mode**: Themed UI for day and night work.
